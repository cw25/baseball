# baseball

## Data

This repo employs data files obtained from retrosheet.org. More info can be found in the project LICENSE.

## Development

Run local desktop version:

* `npm run dev`

Run local web version:

* `npm run build`
* `cd dist`
* `npx http-server`
* Access on http://localhost:8080
